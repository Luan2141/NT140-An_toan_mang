# Network Security Labs

## Lab 1: Getting Comfortable with Kali Linux
- **Objective:** 
  - Familiarize with the Kali Linux environment.
  - Learn the basics of command-line tools and utilities available in Kali Linux.
  - Set up a lab environment for penetration testing.
  - Explore essential tools like `nmap`, `netcat`, and basic scripting.

## Lab 2: Information Gathering
